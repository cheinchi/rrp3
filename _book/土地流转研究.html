2.3 查看变量标签

对于原有数据，都是存在一个标签来显示原始的问题形式，因此我们可以先查看我们想要找的问题的标签是否对应。先创建一个get_var_label的函数。

library(purrr)
get_var_label <- function(dta) {
  labels <- map(dta, function(x) attr(x, "label"))
  data_frame(
    name = names(labels),
    label = as.character(labels)
  )
}

根据观察原有标签，我们可知道fk201_a_n的变量都是拥有的农业资产，fk202_a_n、fk203_a_n和fk204_a_n分别是经营、转租入和转租出多少农业资产。
